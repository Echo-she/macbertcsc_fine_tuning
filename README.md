# MacBertCsc-Fine tuning

#### 微调中文改错模型

> 使用Pytorch+Transformers实现

由于最近自己的需要，使用Transformers库进行了中文纠错的微调模型，并设计了一个GUI程序

训练代码为main.ipynb，在字粒度上面，Accuracy达到了91%的水平

GUI代码为main.py，通过调用保存微调后的模型进行推理
